# Help Team

Простой сайт объявлений для помощи собакам.

## Стек

- PHP 8.5
- MySQL
- RedBeanPHP
- HTML5
- CSS
- Vanilla JS

## Структура

- `public_html/` — публичная часть сайта.
- `app/` — код приложения.
- `bootstrap/` — загрузка приложения.
- `config/` — конфиги.
- `database/` — миграции.
- `resources/` — шаблоны.
- `storage/` — логи, кэш, временные файлы.
